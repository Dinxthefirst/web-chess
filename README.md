# Web Chess

> Chess in the browser

## Building

### Clone the project

```
$ git clone https://github.com/Dinxthefirst/web-chess.git
$ cd web-chess
```

### Build the project

```
$ go build
```

### Running the server

```
$ ./web-chess server
```

### Accessing the website

The server should start on the localhost address of `127.0.0.1:42069`

### Testing the project

```
$ ./web-chess perft-test
```
